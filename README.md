# Maison Dorée — Artisan Bakery

A premium bakery e-commerce site built with Vue 3, Vite, Tailwind CSS v4, and GSAP.

## Stack

- **Vue 3** — Composition API with `<script setup>`
- **Vue Router 4** — Client-side routing with page transition animations
- **Tailwind CSS v4** — Utility-first styling with custom design tokens
- **GSAP** — Page transitions, scroll animations, custom cursor
- **Vite 8** — Build tooling

## Features

- 26 pages including shop, product detail, custom cakes builder, blog, gallery, and more
- Cart and wishlist with persistent state
- Full-screen search overlay with live results
- Custom animated cursor
- Mobile-responsive navigation
- Deployed on Netlify with SPA routing

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
